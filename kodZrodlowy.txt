Utworzenie na github.com nowego i publicznego repozytorium
Pobranie Git for Windows
W terminalu (używam VScode):    
    git --version
    git init
    git remote add origin https://github.com/DawidDymek/devops-projekt.git
    echo "# DevOps Projekt" > README.md
    git add .
    git config --global user.name "Dawid Dymek"   
    git config --global user.email "dawiddymek@onet.pl"    
    git commit -m "Inicjalizacja projektu"
    git push -u origin master
    mkdir app
    cd app
    wewnątrz folderu app dodaje plik index.html a w nim umieszczam kod
    cd ..
    git add . 
    git commit -m "dodanie kodu index.html"
    git push
w folderze głownym tworzę plik : Dockerfile, a w nim zamieszczam kod 
w terminalu :
    docker build -t devops-frontend .
