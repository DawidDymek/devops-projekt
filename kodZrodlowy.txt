Utworzenie na github.com nowego i publicznego repozytorium
Pobranie Git for Windows
W terminalu (używam VScode):    
    git --version
    git init
    git remote add origin https://github.com/DawidDymek/devops-projekt.git
    echo "# DevOps Projekt" > README.md
    git add .
    git config --global user.name "Dawid Dymek"   
    git config --global user.email "dawiddymek@onet.pl"    
    git commit -m "Inicjalizacja projektu"
    git push -u origin master
    mkdir app
    cd app
    wewnątrz folderu app dodaje plik index.html a w nim umieszczam kod
    cd ..
    git add . 
    git commit -m "dodanie kodu index.html"
    git push
w folderze głownym tworzę plik : Dockerfile, a w nim zamieszczam kod . (Włączam docker)
w terminalu :
    docker build -t devops-frontend .
    docker run -d -p 8080:80 --name devops-frontend devops-frontend
w przeglądarce http://localhost:8080 i widzę stronę.
w terminalu: 
    docker stop devops-frontend
    docker rm devops-frontend
    git add Dockerfile
    git commit -m "Dodano Dockerfile"   
    git push
tworzenie workflows w terminalu:
    mkdir -p .github/workflows
następnie tworze w nim plik : .github/workflows/ci.yml
w terminalu : 
    git add .github/workflows/ci.yml
    git commit -m "Dodano workflow CI sprawdzający index.html"
    git push
tworzymy nowy plik docker-compose.yml
dodajemy kod
w terminalu:
    docker compose up -d
    sprawdzenie aktywnych kontenerów: docker ps 
    zatrzymanie aplikacji: docker compose down
    git add docker-compose.yml
    git commit -m "Dodano plik docker-compose.yml do projektu"
    git push
